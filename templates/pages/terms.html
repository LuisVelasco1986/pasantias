Términos y Condiciones de Uso

Última actualización: 2026

Este sistema está destinado a la gestión del control de acceso de personas dentro de las instalaciones de la empresa.

1. Uso del sistema

El sistema es de uso interno y exclusivo del personal autorizado. No está permitido el acceso a personas no autorizadas.

2. Creación de cuentas

Las cuentas de usuario serán creadas por administradores del sistema. Cada usuario es responsable de mantener la confidencialidad de sus credenciales.

3. Responsabilidades del usuario

  • Proporcionar información veraz
  • No compartir credenciales de acceso
  • Usar el sistema únicamente para fines laborales

4. Seguridad

El sistema implementa medidas de seguridad para proteger la información. Sin embargo, el usuario también es responsable del uso adecuado de sus credenciales.

5. Restricciones

No está permitido:

  • Intentar acceder a información no autorizada
  • Manipular el sistema de forma indebida
  • Utilizar el sistema para fines distintos a los establecidos

6. Disponibilidad

El sistema puede no estar disponible en ciertos momentos debido a mantenimiento o fallas técnicas.

7. Modificaciones

La administración se reserva el derecho de modificar estos términos en cualquier momento.

8. Terminación de acceso

El acceso al sistema puede ser suspendido o eliminado en caso de incumplimiento de estos términos.

9. Aceptación

El uso del sistema implica la aceptación de estos términos y condiciones.
